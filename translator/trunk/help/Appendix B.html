Appendix B

Description of menu commands

The following lists describes all menu commands and their usage.

Main Menu

 

Menu Item Shortcut Key Description
File
Open Original Ctrl+O Prompts for the original as well as the translation file and loads them into the listview. If you have previous loaded files that have been modified, you wil be prompted to save before the new file(s) are loaded.
Save Original As... Enables you to save the original file using a different name. Note that you can't edit the original file in Ini Translator so this command is provided only to simplify copying the original file to another location.
Open Translation Ctrl+Shift+O Opens a file in the translation column without reloading the original file
Save Translation Ctrl+S Save the edits to the translation file
Save Translation As... Save the translation to a file with another name
Preferences Alt+Enter Shows the Preferences dialog
Import...   Displayed if any import plugins are installed
Export..   Displayed if any export plugins are installed
Reopen Shows a submenu of recently opened files. The list displays both the original and the translation file
Reopen - Clear Clears the list of recently opened files
Reopen - Remove invalid Removes files from the recently opened files list that are no longer available on the system
Exit Alt+F4 Closes the program. If you have modified the translation file, you will be prompted to save before exiting.
Edit
Undo Ctrl+Z Reverts the last edit change
Cut Ctrl+X Cuts selected text to clipboard
Copy Ctrl+C Copies the selected text to the clipboard
Copy from Original Ctrl+Shift+C Copies the current original text into the translation
Copy from Name Ctrl+Shift+M Copies the current name into the translation
Paste Ctrl+V Paste text from the clipboard into the current location
Find Ctrl+F Display a search dialog where you can type the text to search for
Find Next F3 Continues a search without showing the dialog. If no search is in progress, works the same as Find
Replace Ctrl+H Displays a dialog where you can type the text to search for as well as the text to replace it with
Toggle Focus F6 Toggles focus between the list and the translation editor
Actions
Spelling... F7 Displays the spell check dialog
Previous Ctrl+Up Moves to the previous item in the list
Next Ctrl+Down Moves to the next item in the list
Prev untranslated Ctrl+Shift+U Moves to the previous untranslated item (if there is one)
Next untranslated Ctrl+U Moves to the next untranslated item (if there is one)
Font... Ctrl+Shift+F Shows a dialog that allows you to change the font for the application
Restore original sort Ctrl+Shift+R Restores sorting to the original file order: You can sort the items by clicking on the list columns and later restore the original sort order by pressing Ctrl+Shift+R. The order of items when you save is always in file order regardless of the current sorting in the list.
Trim... Ctr+Shift+T Displays the trim dialog
Toggle Bookmark (0..9) Ctrl+Shift+0..Ctrl+Shift+9 Toggles a bookmark on or off
Goto Bookmark (0..9) Ctrl+0..Ctrl+9 Select item where bookmark is located
Configure Keyboard... Displays a dialog where you can change the shortcut keys for the commands in the program. Shortcuts are saved in a file named translator.alf in the application folder
View
Next unmatched shortcut F8 Jumps to the next item where the presence of a shortcut key ('&') is different between original and translation
Next suspicious translation Shift+F8

Searches for an item that matches any of the following conditions:
1. The number of leading and/or trailing spaces are different between original and translation.
2. Trailing "control" characters (.:?! and the like) doesn't match between original and translation.
3. Original and translation is exactly the same.
4. The translation is empty (same as "Next untranslated item")
Show Quotes Ctrl+Q Encloses strings in quotes when editing them. This aids in seeing whitespace at the beginning and end of the string and also prevents the program from automatically trimming trailing spaces. The quotes are not included in the actual string when it is saved
Show Key Details Ctrl+D Shows a panel with details about the Section and Name of the currently selected item
View Comments Ctrl+M Shows/hides separate window that displays any available comments for the currently selected item
View Orphaned Items Displays a dialog with items found in the translation file that wasn't found in the original file. You can copy an item from the list to the Clipboard by right-clicking an item and selecting "Copy" (or press Ctrl+C).
Tools Shows or hides the Tools toolbar. You can also drag the Tools toolbar off its docking area and click the close button to hide it.
Dictionary
New Dictionary... Ctrl+Alt+N Creates a new or adds new items to the currently loaded dictionary
Load Dictionary... Ctrl+Alt+O Loads a previously saved dictionary
Save Dictionary... Ctrl+Alt+S Saves the currently loaded or created dictionary to a file
Translate using Dictionary Ctrl+Alt+T Translates all matching untranslated (empty) items using the dictionary
Add to Dictionary Ctrl+Alt+A Adds the current Original/Translation pair to the dictionary. If either Original or Translation is empty or the item is already in the dictionary, nothing happens. If the dictionary is inverted, the item is adjusted to match.
Edit Dictionary... Ctrl+Alt+E Displasy the edit dictionary dialog
Invert Dictionary Ctrl+Alt+I Inverts the dictionary, i.e an english - german dictionary becomes a german-english dictionary
Tools
Customize... Displays the External Tools dialog 
Help
Help F1 Shows this help document in the default viewer
About... Displays a dialog with copyright notice, version information and credits


 

Listview Menu

Menu Item Shortcut Key Description
Copy from Original Ctrl+Shift+C Copies the current original text into the translation
Copy from Name Ctrl+Shift+M Copies the current name into the translation
Add to Dictionary Ctrl+Alt+A Adds the current item to the dictionary (for details, see Main Menu description above)
Next untranslated Ctrl+U Moves to the next untranslated item (if there is one)
Prev untranslated Ctrl+Shift+U Moves to the previous untranslated item (if there is one)
Clear all translations Ctrl+Shift+Del Sets all items in the translation to an empty string.
Use this translation everywhere Ctrl+Shift+H Uses the current translation to translate all other matching original items (works like Replace All)
Translated Ctrl+T Indicates whether the item has been translated or not. CLick to toggle indicator on/off.
Add Item... Ctrl+Ins Adds a new item to both original and translation.
Edit Item... Shift+Ins Edit both original and translation of the current item.
Delete Item... Ctrl+Del Deletes the current item in both original and translation.


 

Edit Field Menu

Menu Item Shortcut Key Description
Undo Ctrl+Z Reverts the last edit change
Cut Ctrl+X Cuts selected text to clipboard
Copy Ctrl+C Copies the selected text to the clipboard
Paste Ctrl+V Paste text from the clipboard into the current location
Use this translation everywhere Ctrl+Shift+H Uses the current translation to translate all other matching items even if they are already translated
Select All Ctrl+A Selects all the text in the field
ASCII Value(s) Ctrl+I Displays a dialog with the currently selected text converted to hex and also displays the key sequence needed to enter the text on the numeric keyboard
Dictionary Displays a submenu with the first translation found in the dictionary for the currently selected word in either the Original or the Translation field. If no translation is found, the submenu displays "(none)". Click on the submenu to replace the current selection in the Translation field.
